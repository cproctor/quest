class quest.map.Map¶

Implements a map for a level or stage in the game.

Each QuestGame may have multiple Maps. Each Map represents a game map with multiple layers, allowing the map to describe walls, loot, and multiple layers of background imagery.

background_color¶

a 3-tuple of integers for red, green, blue. Each from 0-255. The arcade.color module also predefines many colors.

tile_scaling¶

Factor by which to scale all map tiles. Default is 1.

class quest.map.MapLayer(name, sprite_list=None, roles=None)¶

Each Map is made up of one or more MapLayers. Each MapLayer contains sprites assigned to particular roles in the game. (The game defines the meaning of roles, but several are built-in: layers with the “display” role will be shown and sprites in layers with the “wall” role will prevent the player from walking into them.

class quest.map.TiledMap(filename, layer_roles)¶

A subclass of Map which loads its layers from a TMX file.
